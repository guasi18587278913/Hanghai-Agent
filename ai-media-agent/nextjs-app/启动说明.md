# 🎨 Next.js 现代化Demo启动指南

## 🚀 快速启动

### 1. 安装依赖
```bash
cd nextjs-app
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

### 3. 访问应用
打开浏览器访问：http://localhost:3000

## 🔧 自定义配置

### 修改颜色主题
编辑 `app/globals.css` 中的CSS变量

### 调整动画速度
修改 `tailwind.config.ts` 中的animation配置

### 添加新问题
编辑 `app/page.tsx` 中的questions数组
